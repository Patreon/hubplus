# HubPlus

A command line tool to manage our pull request flow.


# Installation

1. Install and set up [`hub`](https://github.com/github/hub#installation)

2. Install [`pipsi`](https://github.com/mitsuhiko/pipsi#readme).

3. Install `hubplus`:

        $ pipsi install .


# Usage

To use it:

    $ hubplus --help
